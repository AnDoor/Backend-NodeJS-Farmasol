
# API REST Backend | Farmasol 💊

 Backend en desarrollado con Javascript  para gestionar el inventario de  medicamentos, empleados y sucursales que tiene una farmacia. 
 ```text
  API REST escalable con JS + NODEJS + PostgreSQL.
  ```

   **📁 Estructura del proyecto**

```text
├── config/             # configuracion de BD
├── internal/
├─────internal/handlers # Controladores    
├─────internal/routes   # Rutas y endpoint
├─────internal/services # Logica de negocio  
├── src/                # index.js
  ```
